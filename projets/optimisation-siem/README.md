# Optimisation d'un SIEM : Réduction des Faux Positifs

## Description
Analyse du bruit de fond généré par le SIEM, intégration de flux de **Threat Intelligence** et ajustement des règles de corrélation afin de diminuer drastiquement les fausses alertes et optimiser le temps d'investigation des analystes SOC.
