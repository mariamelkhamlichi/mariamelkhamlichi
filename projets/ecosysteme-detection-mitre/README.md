# Écosystème de Détection Résilient (MITRE ATT&CK)

## Description
Architecture d'un pipeline d'ingestion de logs haute disponibilité avec **Logstash** et **RabbitMQ**. Cartographie de la couverture de détection avec **DeTTECT** et ingénierie de règles d'alerte en temps réel via la stack **ELK** et **ElastAlert**, alignée sur le framework **MITRE ATT&CK**.

## Technologies
`Logstash` `RabbitMQ` `ELK Stack` `ElastAlert` `DeTTECT` `MITRE ATT&CK`
