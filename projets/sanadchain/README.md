# SanadChain

## Description
Conception d'une infrastructure à clés publiques décentralisée (**dPKI**) basée sur la blockchain pour garantir l'intégrité, la traçabilité et la sécurisation des registres fonciers marocains face aux risques de fraude.

## Technologies
`Blockchain` `dPKI` `Cryptographie`
