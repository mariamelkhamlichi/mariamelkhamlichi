# Challenges LetsDefend

Liste des challenges résolus sur LetsDefend. Un fichier par challenge : `nom-du-challenge.md`.

## Template

```md
# [Nom du challenge]

**Catégorie :** (ex. Log Analysis, Malware Analysis, Network Analysis, SIEM...)
**Difficulté :** (Easy / Medium / Hard)
**Date :** 

## Contexte
Description du scénario / des données fournies.

## Démarche d'investigation
Étapes suivies, outils utilisés, requêtes.

## Éléments clés / IOCs
- ...

## Réponse / Conclusion
Résultat du challenge, ce qui a été appris.
```
