## 🎓 Résumé

Le Cyber Kill Chain modélise une cyberattaque en 7 étapes strictement séquentielles : Reconnaissance, Weaponization, Delivery, Exploitation, Installation, Command & Control, Actions on Objectives. Le principe clé : bloquer l'attaquant à n'importe quelle étape stoppe toute la chaîne — plus le blocage intervient tôt, moins les dégâts sont importants. Ce module inclut plusieurs scénarios d'attaque réels (APT38/Lazarus, USB drop, phishing télécom, Cobalt Group) analysés étape par étape pour s'entraîner à situer précisément où en est un attaquant.
