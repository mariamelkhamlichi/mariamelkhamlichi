## 🎓 Résumé

Ce module pose les bases : ce qu'est un SOC (structure, modèles, rôles), les outils clés (SIEM, EDR, Log Management, SOAR, Threat Intelligence Feed) et comment ils s'articulent entre eux, ainsi que les erreurs classiques d'un analyste débutant (confiance excessive dans un seul outil, analyse trop rapide, logs insuffisants, cache VirusTotal ignoré).
