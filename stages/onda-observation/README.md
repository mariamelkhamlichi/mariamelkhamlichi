# Stage d'Observation — ONDA

## Contexte
Stage d'observation réalisé à l'**ONDA** (Office National Des Aéroports), portant sur l'exploration des systèmes d'information critiques.

## Réalisations
- Exploration des SI critiques de planification des vols : **SOAM**, **CADAS**
- Analyse des flux de données sensibles
- Sensibilisation aux impératifs de cybersécurité pour la continuité des opérations

## Technologies / Domaines
`SI critiques` `Aéronautique` `Continuité d'activité`
