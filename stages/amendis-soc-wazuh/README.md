# Stage PFA — Amendis : Déploiement d'un SOC open-source (Wazuh)

## Contexte
Stage de Fin d'Année (PFA) réalisé chez **Amendis**, au sein de la DSI, pour sécuriser les environnements OT (Operational Technology).

## Réalisations
- Déploiement d'un SOC open-source basé sur **Wazuh**
- Virtualisation de l'environnement avec **VMware**
- Configuration centralisée des agents Wazuh
- Mise en œuvre de tests de sécurité : FIM (File Integrity Monitoring), scan de vulnérabilités

## Technologies
`Wazuh` `VMware` `OT Security` `FIM`
